
# Wanderluxe Travel — Static Site

A dark-premium 7-page travel website (HTML/CSS/JS).

## Pages
- `index.html` — Home
- `about.html` — Our story
- `destinations.html` — 8 destinations + FAQ
- `packages.html` — Escape / Signature / Private Jet
- `booking.html` — Reservation form
- `gallery.html` — Slider + photo grid
- `contact.html` — Contact form + studios

## Deploy to Netlify
1. Zip the **contents** of `static-site/` (not the folder itself).
2. Go to https://app.netlify.com/drop and drop the zip.
3. You'll get a live URL instantly.
